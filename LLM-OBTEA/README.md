# LLM-OBTEA · Project Page

Official site for the paper:
**"Integrating Intent Understanding and Optimal Behavior Planning for Behavior Tree Generation from Human Instructions"**

## Project structure

```
.
├── index.html           # Main (and only) page — modern, semantic, framework-free
├── css/
│   └── style.css        # Single stylesheet (custom design system)
├── images/
│   ├── render/
│   │   ├── robot.png        # favicon
│   │   ├── development.png  # hero figure
│   │   └── framework.png    # pipeline figure
│   ├── tree_easy.svg
│   ├── tree_medium.svg
│   └── tree_hard.svg
├── videos/
│   ├── demo_easy.mp4
│   ├── demo_medium.mp4
│   ├── demo_hard.mp4
│   └── A Day at the Café.mp4
└── README.md
```

No build step, no jQuery, no Bootstrap, no Font Awesome — just one HTML file, one CSS file, and a few lines of vanilla JS.
